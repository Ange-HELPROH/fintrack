Vue d'Ensemble du Projet

Description

FinTrack est une application web complète de gestion financière personnelle permettant aux utilisateurs de saisir leurs revenus et dépenses, visualiser les données via graphiques interactifs, recevoir des conseils financiers automatiques et générer des rapports PDF mensuels.

📈 Fonctionnalités Principales

  • Dashboard 4 cartes + 3 graphiques
  • Saisie transactions (26 catégories)
  • Historique filtrable & pagination
  • Conseils financiers auto
  • Reports PDF mensuels
  • Emails de confirmation
  • Thème sombre/clair
  • Design responsive

🎯 Objectifs

  • Partie 1 : App web fonctionnelle sans BDD
  • localStorage pour persistence
  • Validation double (client + serveur)
  • Design moderne & responsive
  • Emails sécurisés via SMTP
  • Partie 2 : MySQL + authentification
